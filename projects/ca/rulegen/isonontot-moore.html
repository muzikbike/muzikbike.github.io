Isotropic Non-Totalistic Rulestring Generator

v0.0.1

This tool generates an isotropic non-totalistic rulestring with the transitions present within corresponding to the boxes which are checked.

Note that the generated rulestrings are not always canonical - partial negation is yet to be implemented.

All good CA simulators will still correct such rulestrings (although you'll still need to manually remove the newlines).

Preferably I'd have this output into a text box (rather than a newline for every single character), and also allow rulestring inputs, but that's for later.
